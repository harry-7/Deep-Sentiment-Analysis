test(test_df, saved_weights=None)¶

Test the trained model and return the test results

This method implements functionality of predicting labels of the test data in the model and evaluate its performance. If saved_weights is provided, then weights loaded from the given path will be used to predict labels

Parameters:
  • test_df – data frame containing testing data
  • saved_weights – path to the saved model if the model has already been trained before
Returns:

TestResults object containing accuracy,f1,precision and recall scores
train(train_df)¶

Train the model and save it to a location if requested

This method implements the functionality of training the given model. The model object is assumed to be a keras model and it is trained accordingly. If a save_path is provided then the weights get saved at the path for the further use.

Parameters:train_df – data frame containing training data
